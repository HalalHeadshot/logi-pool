# Logi-Pool

A rural-first logistics aggregation platform that enables small and marginal farmers to pool produce and coordinate transport using SMS and USSD.

## Phase 1 Scope
- SMS-based produce logging
- Village-level aggregation
- Threshold-based pooling
- Driver notification and assignment

## Tech Stack
- Node.js
- Express
- MySQL
- SMS (simulated)

## How to Run
1. Install Node.js and MySQL
2. Run `npm install`
3. Start MySQL server
4. Run `npm run dev`
