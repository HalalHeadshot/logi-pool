# Logi-Pool

Logi-Pool is a rural-first logistics and resource coordination platform designed for small and marginal farmers using feature phones.  
It enables collective transport and shared access to agricultural equipment using SMS-based workflows.

---

## Phase 1: Logistics Aggregation (COMPLETED ✅)

### Features
- SMS-based produce logging
- Village-level pooling by crop
- Predefined quantity thresholds
- Automatic truck driver notification
- Driver acceptance via SMS
- Idempotent dispatch locking

### Outcome
Enables multiple farmers to share transport and reduce logistics cost without smartphones or internet.

---

## Phase 2: Shared Equipment & Services (COMPLETED ✅)

### Features
- Equipment/service registration via SMS
- Farmer booking via SMS
- Availability locking
- Job completion and release using DONE command
- Reuse of low-bandwidth coordination layer
